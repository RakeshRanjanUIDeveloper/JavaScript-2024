this keyword refers to the object it belongs to, and its value is determined by how a function is called
  • Global Context:this refers to the window object
  • Inside Functions:
    • n non-strict mode, this refers to the global object.
    • In strict mode, this is undefined.
  • Inside Methods: this refers to the object the method belongs to
  • Arrow Functions: They do not bind their own this. They inherit this from their surrounding lexical scope.
  • In Constructors and Classes:this refers to the instance being created.
  • Explicit Binding: We can explicitly set the value of this using call, apply, or bind.
  • Event Listeners: this typically refers to the element that received the event.
