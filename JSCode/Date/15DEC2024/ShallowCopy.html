A shallow copy duplicates the top-level properties of an object or array.

If you modify the top-level properties, the changes won't affect the original.

However, changes to nested structures (objects/arrays) will impact both the copy and the original.

How to do Shallow Copy
  • Using Object.assign()
  • Using the Spread Operator

A deep copy creates an entirely new copy of the object or array, including all nested objects and arrays.

Changes to the copied structure do not affect the original.

How to do Deep copy
  • Using JSON.stringify() and JSON.parse()
