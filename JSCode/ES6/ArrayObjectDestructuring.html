Destructuring is a concise way to extract values from arrays or objects and assign them to variables.
  • Array destructuring allows us to unpack values from an array into individual variables.
    • We can skip elements by leaving a blank space.
    • We can assign default values to variables in case the array element is undefined.
    • Array destructuring can be used to swap values between variables.
  • Object destructuring allows us to extract properties from an object and assign them to variables.
    • We can rename the destructured variables by using a colon
    • We can assign default values if the property is missing or undefined.
    • We can destructure nested objects.
  • Combining Array and Object Destructuring:
    • We can combine both array and object destructuring in one statement.
