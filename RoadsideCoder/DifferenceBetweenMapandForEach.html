map() always returns a new array, where each element is the result of the callback function applied to the corresponding element of the original array.

map() creates a new array by applying a function to each element of the original array. It transforms the elements and returns a new array with the transformed values.


forEach() iterates over each element in the array and executes a provided function once for each element. It doesn't create a new array and is typically used for side effects (such as logging, updating external variables, or modifying elements of the array)

forEach() always returns undefined and does not produce a new array.
